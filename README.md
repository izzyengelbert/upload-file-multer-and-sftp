# Guide

  - Clone this repo.
  - Then from repo directory write these command to start sftp server: 
```sh
$ cd atmoz-sftp
$ docker-compose up -d
```
- Then go back to repo directory:
```sh
$ cd ..
```
- Install npm node modules:
```sh
$ npm i
or
$ npm ci
```
- To run program:
```sh
$ npm start
```
